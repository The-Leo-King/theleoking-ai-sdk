# Security

Do not open a public issue for a suspected vulnerability or include API keys,
customer data, prompts, generated output, or webhook secrets in a report.

Use the private reporting channel published by the API security contract:

https://theleokingai.com/.well-known/security.txt

Supported SDK versions and response commitments follow the current API support
policy at https://theleokingai.com/api/v1/support.
